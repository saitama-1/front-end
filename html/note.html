Note

Tác dụng thẻ meta viewport

        
          <meta name="viewport" content="width=device-width, initial-scale=1.0" />

Hiển thị nội dung trên toàn bộ khung nhìn hiện tại, bình thường trên các thiết bị có màn hình nhỏ thì chữ cũng bị thu nhỏ theo
nên thuộc tính này giúp chữ giữ nguyên kích thước ban đầu, chỉ bị đẩy xuống dòng khi chạm lề màn hình.

Sử dụng các file css khác nhau với các kích cỡ màn hình khác nhau

    
      <link rel="stylesheet" media="screen and (max-width: 600px)" href="small.css" />
      <link rel="stylesheet" media="screen and (min-width: 601px) and (max-width: 1024px)" href="medium.css" />
      <link rel="stylesheet" media="screen and (min-width: 1025px)" href="large.css" />

polyfill

    
      <!--[if lt IE 10]>
      <script src="https://cdnjs.cloudflare.com/ajax/libs/respond.js/1.4.2/respond.min.js"></script>
      <![endif]-->

Để có thể hiển thị trên những trình duyệt cổ như IE < 10

Media querry trên các kích thước màn hình

 
        /* Thiết bị rất nhỏ (điện thoại, 739px trở xuống) */
        @media only screen and (max-width: 46.1875rem) {
        }

        /* Thiết bị nhỏ (máy tính bảng dọc và điện thoại lớn, 740px trở lên) */
        @media only screen and (min-width: 46.25rem) and (max-width: 63.94rem) {
        }

        /* Thiết bị trung bình (máy tính bảng ngang, 1024px trở lên) */
        @media only screen and (min-width: 64rem) {
        }
